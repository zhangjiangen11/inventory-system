CraftStationType
================

**Inherits:** ``Resource``

Determines the type of :ref:`CraftStation<class_CraftStation>`.

.. rst-class:: classref-introduction-group

Description
-----------

This is used by :ref:`CraftStation<class_CraftStation>` to filter the valid :ref:`Recipe<class_Recipe>`\ s in the ``CraftStation.valid_recipes`` list using the value set in the ``Recipe.station`` recipes.
